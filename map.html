General Instructions


☞ This is an Interactive Webmap showing some of the tourist destinations in the state of Victoria, Australia. The map has some basic elements like:

➊ Scale bar at the bottom left.

➋ Clickable Destinations markers with popups and dedicated menu panel.

➌ Zoomin,ZoomOut and home button on the top right.

➍ The Explore Victoria's 'website logo' on the botton right.

➎ Clickable menu buttons: Menu, Fullscreen Mode and About The Map.


☞ HINT: IF THE DESTINATION POPSUPS DOES NOT FIT IN THE SCREEN DOUBLE CLICK THE DISTINATION NAME IN LEFT MENU PANEL.

☞ NOTE: OPEN THE MAP IN YOUR BROWSER'S FULL SCREEN MENU FOR BETTER VIEWING RESULTS.

The web map is not compatible with Internet Explorer web browser
